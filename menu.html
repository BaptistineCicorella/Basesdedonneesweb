A propos

Etudiantes en troisième année d'Administration Economique et Sociale, nous réalisons de site pour le projet de bases de données web. L'objectif de ce projet est de créer une petite base de données accéssible via une interface Web.

Les tables sont créées à l’aide de phpMyAdmin, afin de créer, modifier ou supprimer des données par l’intermédiaire d’une page web.
